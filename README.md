 👨‍🔧 Installation
 ==================
- Install Yarn
- Run ```git clone https://github.com/gitryder/Planit```
- Then run ```cd planit```
- Finally run ```yarn start```
    - Runs the app in the development mode.<br />
    - Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
